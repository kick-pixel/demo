# Todo App

A single-file todo app built with vanilla HTML, CSS, and JavaScript.

## Features

- Add tasks via button click or Enter key
- Toggle completion with animated particle celebration
- Delete tasks individually or clear all completed at once
- Persistent storage via localStorage
- XSS-safe rendering
- Responsive design (mobile + desktop)
- Keyboard accessible

## Getting Started

Open `index.html` in any browser. No build step, no dependencies.
